Shortener api usage




Always use Base url

/create
Parameters:
url: url to be shortened

Optional:
title: title for url (RECOMMENDED)
limit: max amount of clicks for it to be removed (default 10)



Usage:
/create?title=google&url=https://google.com/&limit=10
/     (post to baseurl)
Parameters:
url: url to be shortened

Optional:
title: title for url (RECOMMENDED)
limit: max amount of clicks for it to be removed (default 10)
Content-Type: application/json

example 
                
{
    "title": "google",
    "id": "B0",
    "path": "0xFE6Vhf3h",
    "url": "https://google.com/",
    "shorturl": "http://urls.cripplerick.com/B0/0xFE6Vhf3h",
    "stats": "http://urls.cripplerick.com/stats/B0/0xFE6Vhf3h",
    "clicks": 0,
    "limit": 10
}
